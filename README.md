## Requirement

- yarn 1.22.10

## Installation

1. clone repository
```sh
git clone git@github.com:Futaba-Kosuke/4i-qr-treasure-hunt.git
cd 4i-qr-treasure-hunt
```

2. install modules
```sh
yarn install
```

## Usage

```bash
# Start the server
yarn start
```
